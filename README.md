# api-fastify-vitest-boilerplate

A fastify REST API with vitest

## Getting started

```bash
cp .env.example .env
npm install
npm run db:up
npm run db:migrate
npm run dev
```

## Available Scripts

In the project directory, you can run:

### `npm run dev`

To start the app in dev mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### `npm start`

For production mode

### `npm run test`

Run the test cases.
